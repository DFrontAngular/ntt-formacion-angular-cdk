Layout

El módulo de Laout proporciona utilidades para crear interfaces de usuario responsivas que reaccionan a los cambios en el tamaño de la pantalla.

Muy útil para cuando se necesitan dos diseños totalmente distintos para escritorio/móvil que a través de CSS no serian posibles.

Una ventaja muy importante, ya que no tendriamos que estar ocultando componentes con display none, o similar. Sino que gracias al Layout, los componentes no se pintarían en el HTML, aumentando asi el rendimiento de nuestra aplicación.
