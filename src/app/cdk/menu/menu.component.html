Muy importante @import '@angular/cdk/overlay-prebuilt.css'; en el styles.scss de nuestro proyecto para que el overlay funcione

El @angular/cdk/menu nos ayuda a crear menus personalizables con interacciones badadas en la especificación de WAI ARIA.

El menú @angular/cdk/ está diseñado para ser altamente personalizable según sus necesidades. Por lo que gracias a todas las directivas que tiene se pueden aplicar las clases de CSS correspondiente a cada elemento de html.

  • Directiva: cdkMenu; CSS class: .cdk-menu
  • Directiva: cdkMenuBar; CSS class: .cdk-menu-bar
  • Directiva: cdkMenuItem; CSS class: .cdk-menu-item
  • Directiva: cdkMenuItemRadio; CSS class: .cdk-menu-item-radio
  • Directiva: cdkMenuTriggerFor; CSS class: cdk-menu-trigger
