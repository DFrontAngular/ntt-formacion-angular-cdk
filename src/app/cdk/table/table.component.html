Table

El modulo de CdkTable nos proporciona un data-table totalmente personalizable, con columnas dinámicas y una estructura del DOM accesible. La tabla proporciona una base sobre la que se pueden construir otras funciones, como la ordenación y paginación.

El primer paso para la creación de nuestra tabla es definir los templates del header-row (cdkHeaderRowDef),data-row (cdkRowDef) y el footer-row (cdkFooterRowDef), siendo todos ellos opcionales.

El siguiente paso es definir las filas de la tabla. header-row (cdkHeaderRowDef), data-row (cdkRowDef), y footer-row (cdkFooterRowDef)

El estilizado de las columnas es muy sencillo, ya que cada fila y celda se le proveerá con una clase de css que incluye el nombre de su columna, por ejemplo: cdk-column-weight, además de las clases cdk-row (a cada th) y cdk-cell (a cada td), etc.
