Listbox

El módulo @angular/cdk/listbox proporciona directivas para ayudar a crear interacciones de listbox personalizadas basadas en el patrón de listbox WAI ARIA.

Vienen preparados tanto para añadirles reactive forms, template-driven forms o con value binding

Este módulo nos viene con todos los comportamiento de accesibilidad, incluyendo soporte bidi, interacción con teclado, y gestión de focus. Todas con sus directivas asociadas de ARIA roles en sus host element.

El listbox es altamente personalizable, aplicandose clases de CSS con cada directiva.

  • Directiva: cdkOption; CSS class: .cdk-option (siempre)
  • Directiva: cdkOption; CSS class: .cdk-option-active, si la opcion esta activa.
  • Directiva: cdkListbox; CSS class: .cdk-listbox (siempre)

Además de una lista de atributos aria como: [aria-disabled="true"], [aria-selected="true"], [aria-orientation="vertical"] ...
